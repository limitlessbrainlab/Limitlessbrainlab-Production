1. Platform Overview

NeuroSense360 is a comprehensive, multi-role SaaS platform for EEG-based brain wellness assessment, clinic management, and patient engagement. It combines a public-facing marketing site with role-based dashboards for Super Admins, Clinic Administrators, and Patients.

The platform enables neurofeedback clinics to upload raw QEEG (quantitative EEG) data, process it through AI-powered algorithms, and generate branded brain wellness reports—all with built-in subscription management and payment processing.

Key Metrics Displayed

  • 25,000+ members across the platform
  • 2,500+ registered clinics
  • Multi-currency payment support (8 currencies)
  • AI-powered report generation in ~8-10 minutes

8. User Flows

Flow 1: Public Visitor → Patient

Landing Page → Click "Get Started" → Register (name, email, password, phone, address) → Email confirmation → Login → Patient Dashboard → Complete profile onboarding → Take initial brain assessments → View brain health scores → Access FREE tier content → Upgrade to BASIC/PRO/PREMIUM for more features

Flow 2: Public Visitor → Clinic Registration

Landing Page → Click "Clinic Login" / "Get Started" → Register as clinic (name, contact person, email, phone) → 30-day trial starts (50 free reports) → Clinic Dashboard → Onboarding wizard → Add first patient → Upload QEEG PDFs (Eyes Open + Eyes Closed) → AI pipeline generates report (~8-10 min) → Patient gets notified, views report → Monitor usage → Purchase more reports when low

Flow 3: Patient Report Viewing

Patient Login → Dashboard → Notification: "New report available" → View generated PDF report → See brain wave analysis & scores → Review personalized care plan → Track improvement metrics over time → Schedule follow-up appointment

Flow 4: Super Admin Daily Operations

Admin Login (superadmin@neuro360.com) → Dashboard: check system analytics → Review pending clinic registrations → Approve/reject → Monitor revenue in Payment History → Check report quality in Reports section → Review algorithm processing queue → Manage coach assignments → Update CMS content if needed → Review alerts for system issues

Flow 5: Payment / Subscription Upgrade

User hits usage limit or wants more features → Navigate to Subscription page → View current usage & available packages → Select desired package/tier IF location = India: → Razorpay checkout (INR, UPI, cards, wallets) ELSE: → Stripe checkout (USD/EUR/GBP/AED + cards) → Payment processed → Instant quota increase / feature unlock → Confirmation email + invoice generated

9. QEEG Report Generation Pipeline

  1. Upload Phase
    Clinic admin uploads Eyes Open PDF + Eyes Closed PDF. Files stored in temporary server location via Multer.
  2. Logo Modification
    Original EEG vendor logo removed from PDFs. NeuroSense branding applied. Modified PDFs re-saved with metadata preserved.
  3. Cloud Upload
    Both modified PDFs uploaded to Supabase qeeg-uploads storage bucket. Public access URLs generated for downstream processing.
  4. qEEG Pro Processing
    PDFs parsed to extract EEG data. Brain frequency bands calculated. Brain asymmetry analyzed. Coherence metrics computed. Standardized qEEG report generated.
  5. NeuroSense Algorithm Analysis
    Proprietary algorithms analyze brain wave patterns. Neurological risk assessment performed. Brain health scores calculated. Standardized report generated. Personalized care plan created.
  6. Gemini AI Report Generation
    Clinical insights synthesized from all data. Narrative report generated with recommendations. Care protocols personalized. Content formatted for PDF. NeuroSense branding applied.
  7. Report Finalization
    Final PDF stored in Supabase neurosense-reports bucket. Patient record updated with report link. Audit log entry created. Patient notification sent via email. Reports-used counter incremented for clinic.

Estimated Total Processing Time: 8–10 minutes from upload to final report delivery.

10. Payment & Subscription Flow

Clinic Subscription Model

  1. Registration: 30-day trial period with 50 free reports, no payment required
  2. Trial Warnings: Email alerts at 7 days, 3 days, and 1 day before expiration
  3. Purchase: Choose package (10 / 25 / 50 / 100 reports) with volume discounts
  4. Payment: India → Razorpay (INR, UPI, cards) | International → Stripe (multi-currency)
  5. Activation: Instant quota increase, confirmation email, invoice generated

Patient Subscription Model

  1. Free Tier: Basic brain info, profile, limited assessments, 0 reports
  2. Upgrade: Click "Upgrade" in dashboard, see tier comparison
  3. Payment: Currency auto-detected by location, Stripe or Razorpay
  4. Activation: Features unlock instantly, confirmation email sent
