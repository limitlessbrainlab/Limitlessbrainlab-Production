2. APPLICATION OVERVIEW

Neurosense is an AI-powered neurological assessment platform that enables clinicians to analyze patient brain data, generate clinical reports, and track treatment outcomes using advanced machine learning.

CORE USER FLOW

  • Patients register and access brain wellness data and training protocols
  • Clinicians upload EEG files → Gemini AI generates clinical reports → PDF export
  • Clinics manage patient rosters, subscriptions, and team members
  • Super Admins monitor system health, manage clinics, track analytics
  • Real-time notifications for reports, payments, and clinical alerts

KEY FEATURES

  • QEEG Report Generation: Upload EEG files → AI generates detailed clinical reports → Export as PDF
  • Patient Management: Register, track medical history, assign training protocols
  • Brain Training: Personalized protocols based on assessment results
  • Subscription Management: Clinic subscriptions with Razorpay billing
  • Real-time Notifications: Email alerts for reports, payments, updates
  • Secure Data Access: Row-level security ensures privacy

5. THIRD-PARTY INTEGRATIONS

Google Gemini API

  • Purpose: Analyze EEG files and generate clinical reports
  • Model: Gemini 2.5 Flash (vision capabilities)
  • Rate: ~20 requests/day | Cost: ~$0.04 per request
  • Error Handling: Automatic retry with exponential backoff

Supabase (Database + Auth + Storage)

  • Database: PostgreSQL with all user data and records
  • Auth: Supabase Auth for user login/signup
  • Storage: PDF reports in patient-reports bucket
  • Security: Row-Level Security (RLS) enabled on all tables

Razorpay (Subscription Billing)

  • Subscription Types: Basic, Pro, Enterprise (monthly/annual)
  • Payment Flow: Create order → User pays → Verify signature → Update DB
  • Security: PCI-certified, no card data stored

Nodemailer (Email Delivery)

  • SMTP: Gmail with app-specific password
  • Emails: Reports, credentials, notifications
  • Rate: ~100 emails/min from Gmail

6. DEPLOYMENT GUIDE

FRONTEND (Vercel)

  • Auto-deployment on push to main branch
  • Build: npm run build (Vite)
  • Environment variables: Set in Vercel dashboard
  • URL: https://limitlessbrainlab-eight.vercel.app

BACKEND (Render.com)

  • Auto-deployment on push to main branch
  • Runtime: Node.js 18.x
  • Start: cd server && npm start
  • ⚠ CRITICAL: GEMINI_API_KEY must be in render.yaml with sync:false
  • URL: https://neurosense-api.onrender.com

DATABASE (Supabase)

  • Create tables: Run database/schema.sql in SQL editor
  • Enable RLS: Row-Level Security on all tables
  • Backups: 7-day retention with automatic daily backups
  • Connection: PostgreSQL via Supabase URL
