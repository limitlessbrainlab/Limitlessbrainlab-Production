Deployment Summary
Issues resolved and changes deployed to staging on 30 May 2026
Staging Build
Changes Deployed This Session
  • Runtime Crash Fix — Algorithm Data Processor
    Resolved a hard crash on the Algorithm Data Processor page where outputData.slice().map() was called on a JSON string instead of a parsed array. Added a parseResultsData() guard that safely parses doubly-encoded JSONB data from the algorithm_results table. Applied across all 6 call-sites.
    Bug Fix Frontend AlgorithmDataProcessor.jsx
  • API Proxy Fix — Vercel to Render Backend
    All /api/* requests from the Vercel frontend were hitting Vercel's own CDN instead of the Render backend, returning index.html and causing 404 errors on all email and report endpoints. Fixed by adding an API proxy rewrite in vercel.json that forwards /api/:path* to https://neuro360-backend.onrender.com/api/:path*.
    Bug Fix Deployment Config vercel.json
  • NEXAPROC_MASTER_KEY — Render Environment Variable
    The NEXAPROC_MASTER_KEY server environment variable was missing from the Render dashboard, causing the "Upload to Claude" (Claude Report generation) feature to fail with an authentication error. Variable added to Render with sync: false as required. Render auto-redeployed the backend after save.
    Config Render Backend nexaprocService.js
  • GitHub Repository Access — Push Permissions
    Resolved push permission issue on limitlessbrainlab/Limitlessbrainlab GitHub repository. Added collaborator write access for the deployment account and accepted the pending repository invitation via the GitHub API.
    Config GitHub
  • Vercel Production Deployments
    Two production deployments pushed to https://limitlessbrainlab-eight.vercel.app: first for the Algorithm Processor crash fix, second for the API proxy / email fix. Both builds completed successfully (build time ~17s each).
    Deploy Vercel
